## ğŸ¯ Problem Statement

Electronic waste (e-waste) is one of the fastest-growing waste streams globally. In India alone, over **3.2 million tonnes** of e-waste is generated annually, with only **10-15%** being properly recycled.

### Challenges:
- âš ï¸ Lack of awareness about e-waste collection centers
- âš ï¸ No incentive for proper disposal

## ğŸ“¦ Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/Samarthjadhavsj/E-Waste-Management-System.git
cd E-Waste-Management-System
```

### Step 2: Install Dependencies

```bash
npm install
```

### Step 3: Set Up Database

#### Option A: Using XAMPP
1. Start XAMPP Control Panel
2. Start MySQL service
3. Open phpMyAdmin: `http://localhost/phpmyadmin`
4. Create database: `ewaste_locator`

#### Option B: Using MySQL CLI
```bash
mysql -u root -p
CREATE DATABASE ewaste_locator;
EXIT;
```

### Step 4: Configure Environment Variables

Create a `.env` file in the root directory:

```env
# Database Configuration
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=ewaste_locator

# JWT Secret
JWT_SECRET=your_super_secret_jwt_key_here_change_in_production

# Server Configuration
PORT=3000
NODE_ENV=development
```

### Step 5: Seed Database

```bash
# Seed products
node scripts/seedProducts.js

# Create admin account
node scripts/createCustomAdmin.js
```

### Step 6: Start the Server

```bash
npm start
```

ğŸ‰ **Server running at:** `http://localhost:3000`
